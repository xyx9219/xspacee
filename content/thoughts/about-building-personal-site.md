## 为什么需要个人网站

社交媒体时代，我们习惯于在各种平台上发布内容。但这些平台终究不属于我们自己——算法决定谁能看到我们的内容，平台规则随时可能改变，甚至账号也可能在一夜之间消失。

一个**完全由自己掌控**的网站则不同：

- 你可以自由决定它的样子
- 你的内容不会被算法过滤
- 它不会因为某个公司的决策而消失
- 它是你数字身份最真实的体现

## 设计哲学

我的设计灵感来源于那些充满**实验精神**的网页作品：

1. **大胆的排版** — 让文字本身成为视觉元素
2. **强烈的色彩对比** — 深色背景配合亮色点缀，营造沉浸感
3. **几何元素的穿插** — 圆形、方形、线条构成视觉节奏
4. **代码与设计的融合** — 作为开发者，代码是我表达的一部分

## 技术选择

这个网站完全使用**原生技术**构建：

- HTML5 语义化标签
- CSS3 自定义属性、Grid、Flexbox
- Vanilla JavaScript（无框架依赖）
- Canvas API 粒子动画
- Markdown + Python 构建脚本

没有使用 React、Vue 或任何 UI 框架。有时候，**最简单的工具反而能带来最纯粹的表达**。

## 持续生长

这个网站不会是一成不变的。随着我的学习深入，它会不断进化。新的交互效果、新的排版尝试、新的内容板块……

它是我的一面镜子，映照着我的成长轨迹。

> "The web is still a young medium. We have so much left to explore."
